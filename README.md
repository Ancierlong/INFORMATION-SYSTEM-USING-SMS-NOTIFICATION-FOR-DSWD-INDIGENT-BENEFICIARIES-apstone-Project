Capstone Project: POPULATIONALYSIS: INFORMATION
SYSTEM USING SMS NOTIFICATION FOR DSWD
INDIGENT BENEFICIARIES

- Centralized Beneficiary Registry (CRUD with File Uploads):
A robust administrative portal to Create, Read, Update, and Archive/Delete comprehensive profiles of indigent families and individuals.
Integrated document management allowing caseworkers to securely upload and link essential requirements (e.g., Certificates of Indigency, government-issued IDs, and intake sheets) directly to the beneficiary’s profile.

- Targeted SMS Announcement Engine:
An manual SMS broadcasting module used to send real-time notifications, payout schedules, community program announcements, and disaster relief updates directly to beneficiaries' mobile numbers.
Supports contact categorization (e.g., sorting by Barangay or specific assistance programs) to ensure announcements only reach the relevant recipients.

- Population Analysis & Reporting:
A secure dashboard offering administrative analytics on demographic distribution, household sizes, and aid allocation trends.
Generates exportable compliance reports to assist DSWD officers in data-driven decision-making and resource distribution.
